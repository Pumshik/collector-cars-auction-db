# Аукционная площадка коллекционных автомобилей (PostgreSQL)

Реляционная база данных для платформы по продаже коллекционных автомобилей. Проект включает проектирование схемы, бизнес-логику на уровне БД и аналитические инструменты.

## Содержимое

| Файл | Назначение |
|------|------------|
| `Creating.sql` | Создание таблиц (BRAND, MODEL, CAR, COLLECTOR, AUCTION_LOT и др.) |
| `Filling.sql` | Заполнение таблиц тестовыми данными |
| `Advanced_part.sql` | Триггеры, хранимые функции, представления и индексы |
| `Select.sql` | Аналитические запросы (10 запросов) |
| `*.csv` | Дампы данных таблиц |

## Схема базы данных

- **BRAND** : бренды автомобилей (Ferrari, Lamborghini, Porsche...)
- **DESIGNER** : дизайнеры (Marcello Gandini, Gordon Murray...)
- **MODEL** : модели с характеристиками (двигатель, мощность, класс)
- **MODEL_DESIGNER** : связь моделей и дизайнеров с указанием части дизайна
- **COLLECTOR** : коллекционеры (Jay Leno, Ralph Lauren...)
- **CAR** : конкретные автомобили с пробегом, состоянием, цветом
- **AUCTION_LOT** : аукционные лоты со статусами (Active, Sold, Passed)

## Триггеры и функции

- **Автоматическая смена владельца** : при статусе `Sold` обновляется `current_collector_id`
- **Защита от скручивания пробега** : запрет уменьшения `mileage`
- **Индекс премиальности цены** : насколько цена машины выше/ниже средней по рынку
- **Ценовой индекс дизайнера** : сравнение средних цен машин дизайнера с рынком
- **Индекс эффективности л.с.** : стоимость одной лошадиной силы относительно класса

## Представления

- `ACTIVE_AUCTIONS` : текущие активные лоты с полной информацией
- `BRAND_MARKET_STATS` : материализованное представление со статистикой по брендам

## Индексы

- `IDX_AUCTION_STATUS` : ускорение фильтрации по статусу лота
- `IDX_CAR_MODEL_ID` : ускорение JOIN между CAR и MODEL
- `IDX_BRAND_NAME` : ускорение поиска по названию бренда

## Примеры аналитических запросов

- Модели с мощностью выше средней
- Все автомобили конкретного коллекционера
- Выручка по странам брендов
- Итальянские дизайнеры с полным дизайном моделей
- Отклонение цены продажи от ожидаемой
- Средняя стоимость л.с. по классам автомобилей
- Коллекционеры, выступающие и продавцами, и покупателями
